# InstiCloud

Server for storing data in decentralized manner using [GUN](https://gun.eco/)

## Installation

Use the package manager [npm](https://nodejs.org/en/download/) to install the dependencies.

```bash
npm install
```

## Usage
Run the following command to start server.
```bash
node index.js
```
